# Onakice Tools Hub

ชุดเครื่องมือออนไลน์สำหรับเพิ่ม Productivity ในการทำงานด้าน Content, Data Processing และ AI Prompt Engineering

🌐 Live Demo: https://onakice.github.io/

## Overview

Onakice Tools Hub เป็นเว็บไซต์ที่รวบรวมเครื่องมือขนาดเล็ก (Micro Tools) สำหรับช่วยลดงานซ้ำ ๆ ที่ต้องทำในชีวิตประจำวัน ไม่ว่าจะเป็นการจัดรูปแบบข้อความ สร้าง UUID สแกนข้อความ หรือเตรียม Prompt สำหรับใช้งานกับ AI

ปัจจุบันประกอบด้วยเครื่องมือหลักดังนี้

### 📄 Text Scanner

เครื่องมือสำหรับสแกนและวิเคราะห์ข้อความ

**Features**

* ตรวจจับข้อมูลจากข้อความ
* ค้นหา Pattern ที่กำหนด
* รองรับการประมวลผลข้อความขนาดใหญ่

---

### ✨ Content Formatter

เครื่องมือช่วยจัดรูปแบบ Content

**Features**

* จัดรูปแบบข้อความอัตโนมัติ
* ปรับโครงสร้างให้อ่านง่าย
* เหมาะสำหรับ Content Creator และ Marketing

---

### 🆔 UUID Generator

เครื่องมือสร้าง UUID

**Features**

* Generate UUID ได้ทันที
* รองรับการ Copy ไปใช้งาน
* ใช้งานง่ายผ่าน Browser

---

### 🔍 Extractor Merger

เครื่องมือสำหรับ Extract และ Merge ข้อมูล

**Features**

* ดึงข้อมูลจากข้อความ
* รวมข้อมูลจากหลายแหล่ง
* ลดงาน Manual Processing

---

### 😀 Emoji & Dot

เครื่องมือช่วยจัดการ Emoji และ Bullet Point

**Features**

* เพิ่ม Emoji ให้ข้อความ
* จัดรูปแบบ Bullet List
* เหมาะสำหรับ Social Media และ Documentation

---

### 🎓 Senior Course Consultant Tool

เครื่องมือสร้าง Prompt สำหรับวิเคราะห์ Requirement และแนะนำคอร์สเรียน

**Features**

* สร้าง Prompt Template อัตโนมัติ
* ช่วยวิเคราะห์ความต้องการของผู้เรียน
* จัด Roadmap การเรียนรู้
* รองรับการ Copy Prompt ไปใช้งานกับ AI

---

## Local Development

Clone Repository

```bash
git clone https://github.com/Onakice/Onakice.github.io.git
```

เข้าโฟลเดอร์โปรเจกต์

```bash
cd Onakice.github.io
```

เปิดไฟล์

```bash
index.html
```

หรือใช้ Local Server

```bash
npx serve .
```
